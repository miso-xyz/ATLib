  - [x] URL
  - [x] Creation & Current Names
  - [ ] Musics
  - [ ] Wall Comments
  - [ ] Favorites
  - [ ] Albums
  - [ ] Network
- Music
  - [x] Illustration
  - [x] Name
  - [x] BPM
  - [x] Likes
  - [x] Plays
  - [x] Keywords
  - [x] Track Duration
  - [x] Track URLs (Twitter, SWF etc...)
  - [x] Number of Comments
  - [x] Number of Downloads
  - [x] Creation, Last Modified, Upload & Published Dates
  - [ ] Comments
  - [ ] Assets
  - [ ] Remixes
  - [ ] Listeners
  - [ ] In Albums
